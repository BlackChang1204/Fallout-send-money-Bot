## 主要特色、功能
```  
1.可以不用開遊戲輸入指令&聊天
2.透過私訊Bot來執行指令(須設定白名單)  
3.白名單指令系統(避免有心人士亂搞)  
4.自動接受tpa & tpahere(需白名單)
5.直接透過Bot回覆訊息
6.支援透過DC回覆遊戲內訊息
```
